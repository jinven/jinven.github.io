Sass 是对 CSS 的扩展，允许使用变量、嵌套规则、 mixins、导入等功能， 完全兼容 CSS 语法。

Sass 有两种语法：

  • 第一种被称为 SCSS (Sassy CSS)，是一个 CSS3 语法的扩充版本
